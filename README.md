# BuchtauschDatenbank

Dieses Repository enthält die Implementierung einer relationalen Datenbank für eine Buchtausch-Anwendung. Die Datenbank wurde im Rahmen eines Moduls der IU entwickelt und basiert auf PostgreSQL.
Die Anwendung ermöglicht die Verwaltung von Nutzern, Büchern, Buchexemplaren, Ausleihen, Bewertungen sowie Versand- und Abholinformationen. Zusätzlich wurden Views, CHECK-Constraints und Indexe implementiert, um die Datenintegrität und Performance zu verbessern.

Verwendete Technologie:
- PostgreSQL
- DBeaver
- SQL
